# 애견 미용 예약 서비스

- 애견미용 예약 서비스를 구현해보았습니다. 예약, 검색, 조회 기능을 가지고 있습니다.
<br />

### 사용언어 및 개발환경

- Window ,VSCode, Github
- React(CRA), Typescript, TailwindCSS, Axios, Redux-toolkit
<br />
